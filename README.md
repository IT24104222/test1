# MERN Item Manager Dash
A full-stack inventory management application.

## 🚀 Features
* **Full CRUD:** Create, Read, Update, and Delete items.
* **Search:** Real-time filtering of inventory.
* **Colorful UI:** Category-based styling and Glassmorphism.
* **Database:** Powered by MongoDB Atlas.

## 🛠️ Tech Stack
* **Frontend:** React, Axios, CSS3
* **Backend:** Node.js, Express
* **Database:** MongoDB
